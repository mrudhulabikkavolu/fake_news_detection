**PROJECT TITLE:**
**Fake News Detection using Naive Bayes**

**DESCRIPTION:**
Fake news has become a serious issue in today's digital world. Many false news articles spread quickly through social media and online platforms. Detecting such misinformation manually is difficult. Machine learning can help automatically identify whether a news article is real or fake.

This project implements a Fake News Detection system using the Naive Bayes classification algorithm. The model analyzes the text content of news articles and predicts whether the news is real or fake.

The project was implemented using Python in Google Colab and uses libraries such as pandas, numpy and scikit-learn for data processing and machine learning.

**OBJECTIVE:**
The main objective of this project is to build a machine learning model that can classify news articles as real or fake based on their textual content.

**SOFTWARE REQUIREMENTS:**
Operating System: Windows 10 / Windows 11
Programming Language: Python
Platform: Google Colab
Libraries Used: pandas, numpy, scikit-learn, matplotlib

**METHODOLOGY:**
1. Data Collection
A dataset containing news articles labeled as real or fake is used for training the model.

2. Data Preprocessing
The dataset is cleaned and prepared by removing unnecessary characters, converting text to lowercase and preparing it for analysis.

3. Feature Extraction
The textual data is converted into numerical form using TF-IDF (Term Frequency – Inverse Document Frequency) vectorization so that it can be processed by the machine learning algorithm.

4. Model Training
The processed data is used to train a Multinomial Naive Bayes classifier which is suitable for text classification tasks.

5. Model Evaluation
The model performance is evaluated using accuracy score and confusion matrix.

**OUTPUT:**
The trained model takes a news headline or article as input and predicts whether it is Real News or Fake News.

**APPLICATIONS:**
- Detecting misinformation on social media
- News verification systems
- Supporting fact-checking organizations
- Educational use for machine learning projects

**FUTURE ENHANCEMENTS:**
The project can be improved by using advanced machine learning or deep learning models and by developing a web interface for real-time fake news detection.
